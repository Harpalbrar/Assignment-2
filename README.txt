# Resume Webpage

## Description
This project consists of a simple webpage displaying a resume. It's created using HTML for the structure and content, and CSS,js.

## Instructions
To view the webpage locally, follow these steps:

1. Download the project files: 
   - Click on the green "Code" button on the GitHub repository page.
   - Select "Download ZIP" to download the project files as a zip archive.
   - Extract the contents of the zip file to a directory on your local machine.

2. Navigate to the project directory: 
   - Open the extracted folder containing the project files.
   - Locate the `index.html` file within this directory.

3. Open the HTML file in a web browser: 
   - Right-click on the `index.html` file.
